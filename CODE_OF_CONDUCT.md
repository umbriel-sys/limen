# Code of Conduct

## Our Pledge

We are committed to making participation in Limen a respectful and professional experience for everyone.

We expect all participants to:

- Be respectful and considerate in communication
- Engage in constructive, technical discussion
- Focus on improving the project and helping others

## Expected Behaviour

- Use clear, professional language
- Be open to feedback and differing technical perspectives
- Assume good intent from others
- Keep discussions focused on the work

## Unacceptable Behaviour

- Harassment, discrimination, or personal attacks
- Trolling, insulting, or inflammatory comments
- Public or private abuse of any kind
- Repeatedly derailing discussions away from the project

## Scope

This Code of Conduct applies to:

- GitHub issues and pull requests
- Discussions and project communication channels
- Any interaction representing the Limen project

## Enforcement

This project is currently maintained by a single developer.

Unacceptable behaviour may result in:
- Removal of comments or contributions
- Temporary or permanent restriction from participation

## Reporting

If you experience or witness unacceptable behaviour, please report it:

- **Email:** arlo@umbriel-systems.com

All reports will be handled confidentially.

## Notes

Limen is an early-stage project. The focus is on building a high-quality technical system.

Respectful, technically grounded discussion is strongly encouraged.
